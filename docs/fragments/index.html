PROFILE

志來紗衣華 （作詞 / 歌）

ボーカルをRiryka氏に師事。PCゲーム主題歌でデビュー。

Mad Pierrot Labelを拠点に様々なアーティストとのコラボが進行中。作詞も手がける

主な作品 : ルネソフト「ようこそ！スケベエルフの森へ」主題歌「VOICE」歌唱、ルネTeamBitters「それでも妻を愛してる２ -女教師妻・茉莉花の場合-」主題歌「ギルト - guilt -」歌唱 / 作詞、BISHOP 「屈辱」主題歌「DOLL」歌唱 / 作詞

仲村洸祐 （作編曲 / Guitar / Bass）

2017年4月に志來紗衣華と共に音楽同人サークル “Luna Crescente” を結成。作曲、音楽理論を北田陽一郎氏に師事。

Luna Crescente 1st single “dischage”, 2nd single “detection” とロックテイストな楽曲を制作してきた。

“Never Again” ではピアノサウンドとロックサウンドを掛け合わせ、切ない恋愛曲に仕上げた。

鈴木ルヒカ （作編曲）

ゲーム、アニメ音楽ブランド“Angel Note”の代表プロデューサー。

2004年から今日まで才能溢れるAngel Noteメンバーと共に、1000曲を超える楽曲をプロデュース&リリース。

「M3-2017秋」より、志來紗衣華の魅力を伝える様々なタイプの楽曲をリリース。今回はエレクトロポップなナンバーを自ら書き下ろした楽曲でプロデュース！（Tr.2）

 

作品:Angel Noteのクレジットがある全作品。

アニメ:ヴァンカード、ミルキィホームズ等多数。

ゲーム:ゆずソフト、Lump of Sugar、あっぶりけ、暁WORKS、でぼの巣製作所、ALcot、ソフパル、キャラ…他多くのゲームメーカーのゲームタイトル。

歌:Angel Noteの歌手陣をはじめたくさんの人気歌手、人気声優の主題歌やキャラソンをリリース。

Angel Note全アルバム、多メーカーのサントラ盤などCD多数。

 ユージ・ナイトー （作編曲 / Guitar）

2000年に同級生の山田屋カズ（作編曲 / Bass / Engineer）と音楽制作チーム「BAL」を結成。

BasiL “それは舞い散る桜のように(2002年)”のBGMと挿入歌「beloved 〜桜の彼方に」を手掛け、本格的に活動開始。

主な作品：すたじお緑茶 “片恋いの月(2007年)” 主題歌「refrain moon / AiRI」“恋色空模様(2010年)”主題歌 「二人色 / Duca」等。
